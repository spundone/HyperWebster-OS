# distro-tools - maintenance shortcuts

Practical utilities surfaced in one fuzzel menu:

| Item | Opens |
|------|-------|
| System monitor | `caelestia toggle sysmon` / btop |
| Btrfs snapshots | `hyperwebster-snapshots` |
| Monitor profiles | `hyprmoncfg` TUI |
| Kernel manager | `cachyos-kernel-manager` |
| System update | `hyperwebster-update` |
| Maintenance menu | `hyperwebster-maint` |

## Keybind

`Super+Ctrl+Shift+M` opens the maintenance menu.

## CLI

```sh
hyperwebster-maint
```
