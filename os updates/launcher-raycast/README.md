# launcher-raycast — Raycast-like search overlay

Merges keyboard-first launcher defaults into `~/.config/caelestia/shell.json`:

- Fuzzy app search (`useFuzzy.apps`)
- Calculator/actions fuzzy matching
- `maxShown: 8` results

Super+Space still uses the `launcher-fix` drawers IPC bind (reliable open/keep-open).

## Apply / refresh

```sh
hyperwebster-launcher-raycast
```

Pair with `hyperwebster-blur-toggle enable` for frosted glass panels.
